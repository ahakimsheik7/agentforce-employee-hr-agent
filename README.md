# Agentforce Employee HR Assistant Agent

## Overview

This project demonstrates how to build an **Employee AI Assistant using Salesforce Agentforce**.

The goal is to help employees quickly access HR-related information without needing to contact HR staff directly.

The AI agent can answer common questions about:

• Employee benefits
• PTO policy
• HR contact information
• Internal company resources

This project is part of my **Salesforce AI Agent Portfolio**.

---

## Architecture

Employee
↓
Agentforce Employee Agent
↓
Topics (HR Benefits, PTO Policy, HR Contact)
↓
Actions (Retrieve knowledge, query records)
↓
Salesforce Data / Knowledge Base

## Agent Capabilities

The AI Employee Agent can:

• Answer HR-related questions
• Provide benefits information
• Explain PTO policies
• Direct employees to HR contacts

Example questions:

* What benefits does the company offer?
* How many PTO days do employees get?
* How do I contact HR?

## Topics Implemented

### Employee Benefits

Helps employees understand company benefits and insurance coverage.

### PTO Policy

Provides information about vacation and leave policies.
